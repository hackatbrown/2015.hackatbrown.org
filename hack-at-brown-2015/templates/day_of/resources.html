Sponsor APIs

Want to build an app that uses Facebook data, calls an Uber, or works with Kinect? That's what APIs are for — use them to interact with other technologies and build exciting things.

Our sponsors have tons of amazing APIs that you can (but don't have to) use in your project. If you've got any questions, just ask one of the company's own engineers.

  • Brown APIs

    Build an app or website with menus for the Ratty, V-Dub or other Brown eateries using the Brown Dining APIs.

  • Oracle

    Use the Java SDK — there's tons documentation to make learning Java easier.

  • Oracle

    Use the Java SDK — there's tons documentation to make learning Java easier.

  • Microsoft

    Use a Microsoft API like Bing, OneDrive, Office or OneNote. Or create a Windows Universal app. Do amazing things with Kinect, or host your web app on Azure.

  • Two Sigma

    Use the web-based Beaker API to research, visualize and analyze data in several languages.

  • Dropbox

    Use the Datastore API to easily sync data like contacts, to-do items, and game state across multiple devices; or use the Sync API to sync and notify you of remote changes to a user's Dropbox so your app can respond instantly.

  • Facebook

    Use Parse as an easy backend for your mobile or web app, or let users sign into your app using Facebook Login.

  • MongoDB

    Use MongoDB as a database for your app.

  • TripAdvisor

    Use the TripAdvisor API to search attractions, hotels, and restaurant reviews by location.

  • Akamai

    Akamai offers tons of APIs for advanced networking.

  • Lyft

    Use the Lyft API to get estimated travel times, costs, and find nearby drivers. It's top-secret, but Hack@Brown attendees can get a key by emailing Liz (liz@lyft.com) or Ashley (ashley.lorden@lyft.com).

  • Bowery

    Use Bowery to set up and keep your development environments in sync.

  • Sailthru

    Use the Sailthru API for simple solutions to sending email and managing email marketing campaigns.

  • Swipely

    Use Swipely to understand customers and grow sales.

  • Uber

    Use the Uber API to call cars.

  • Tessel

    Build a hardware hack with Tessel. Tessel is programmed in JavaScript, and comes with Wifi built in. Capabilities such as sensing, and actuation can be added to Tessel with plug-and-play modules, or you can do your own wiring through the GPIO bank.

  • Myo

    Integrate Myo into your Android, iOS, Windows and Mac apps.

  • Spark.io

    The Spark Core is a wi-fi development kit for internet connected hardware. It’s the “brains” for a hardware project. The Core has a microcontroller which is a low cost/powered computer that can run a single application and tells the Core what to do.
